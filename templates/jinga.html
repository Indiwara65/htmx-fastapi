Escaping

Tricking the template rendering engine not to render anything inside this block.

{% raw %} {% if variableOne is lower %}

{{variableOne is lowercase}}

{% else %}

{{variableOne}} is not lowercase

{% endif %} {% endraw %}

Above lines are not redereded as they are inside a {% raw %} {% endraw %} block
